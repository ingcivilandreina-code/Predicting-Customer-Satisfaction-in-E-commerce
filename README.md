Business case:
TERRENAL GYM es una cadena ficticia de gimnasios que busca mejorar la retención de clientes identificando de forma anticipada qué miembros tienen mayor probabilidad de cancelar su suscripción.

En este proyecto se desarrolla un flujo completo de Ciencia de Datos para analizar el comportamiento de los clientes y construir modelos de Machine Learning capaces de predecir el abandono (customer churn), facilitando la toma de decisiones orientadas a la fidelización.

Objetivos del proyecto:
* Preparar y transformar los datos mediante un proceso ETL.
* Analizar el comportamiento de los clientes y sus pedidos mediante EDA.
* Identificar patrones y segmentos de clientes mediante técnicas de clustering.
* Entrenar y comparar modelos de Machine Learning para predecir el churn.
* Evaluar el desempeño de los modelos e interpretar sus resultados.

Lenguaje y librerias usadas: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

Metodologia aplicada: 
* ETL y preparación de datos.
* Análisis Exploratorio de Datos (EDA).
* Feature Engineering.
* Entrenamiento y evaluación de modelos de Machine Learning.

Conclusiones: Este proyecto integra un flujo completo de Ciencia de Datos para identificar los factores asociados al abandono de clientes y desarrollar modelos predictivos que apoyen la toma de decisiones y el diseño de estrategias de retención.
